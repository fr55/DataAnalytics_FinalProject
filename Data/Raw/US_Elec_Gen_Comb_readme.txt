# EPA State and County Emission Summaries


## Summary
This dataset was prepared for the final project of Environmental Data Analytics (ENV 872L) at Duke University, Spring 2019

The dataset contains data from State and county summaries PM2.5 emissions.

## Database Information
Data were collected from EPA . More information can be found here: https://www3.epa.gov/air/emissions/where.htm

The following selections were made: 
* Top PM2.5 emitting sectors

Data were accessed 2018-04-05.
